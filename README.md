# garden

my fork of [quartz](https://jzhao.xyz)

## setup

1. `npm i`
2. `ln -s ~/share/garden ./content`

### serve (local dev)

`npm run quartz build --serve`

### deploy

1. `npm run quartz build`
2. deploy `./public`
